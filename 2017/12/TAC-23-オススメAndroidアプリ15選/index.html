[TAC:23] オススメAndroidアプリ15選

はじめに

この記事はtaniho Advent Calendar 2017の23日目の記事です．

今日は僕が常用するオススメAndroidアプリを15個紹介しようと思います．どれも厳選の上使用しているので，是非皆さんも使ってみてください．では早速始めます．

紹介

K-9 Mail

オススメのメーラです．複数のアカウントを色分けして管理できるので便利です．IMAPとPOP3を併用しているのですが，両方問題なく使えています．

K-9 Mail
開発元：K-9 Dog Walkers
価格：無料
平均評価：4.2（合計87,080件）
ダウンロード：500万回以上
（2017年12月23日時点）

Sleipnir

オススメのブラウザです．マルチタブブラウザですが，Google Chromeとは異なり，下にタブバーがあります．ここをタップすることでタブを切り替えたり，フリック操作でタブを閉じることが出来ます．この快適さに惚れてiPhoneでもAndroidでも愛用しています．

Sleipnir Mobile - ウェブブラウザ
開発元：Fenrir Inc.
価格：無料
平均評価：3.9（合計9,407件）
ダウンロード：50万回以上
（2017年12月23日時点）

biyontter

言わずと知れたツイ廃向けTwitterクライアントです．マルチアカウント，マルチタブ対応，リスト表示対応です．ワンタップでfav，RT，引用RT，リプライなどの操作をすることができて非常に(慣れれば)快適です．
デザインの設定も自由に出来たり，複雑なミュート設定も出来たり，アカウントに対応したカラーラベルを付けたり，過去のTLを遡ったりと，普通に使う分にも非常に便利でオススメです．

びよーんったー
開発元：ABS104a
価格：無料（広告を含む）
平均評価：4.1（合計1,727件）
ダウンロード：10万回以上
（2017年12月23日時点）

DigiCal

カレンダーはこのアプリを使っています．気に入ったので広告なしの有償版を購入しました．
複数のカレンダーを色分けして表示することが出来，またカレンダーの表示スタイルも複数の中から選ぶことができるので便利です．

一番便利なのは，ちょうどいい感じのウィジェットがあることです．全体のスケジュールをパッと確認できるので重宝しています．

DigiCal 日本カレンダースケジュール
開発元：Digibites
価格：無料（アプリ内購入あり）（広告を含む）
平均評価：4.4（合計125,462件）
ダウンロード：500万回以上
（2017年12月23日時点）

OpenSync

カレンダーと連絡先の同期に使用しています．僕は自前のサーバにカレンダー・連絡先用サーバを立ち上げているので，専用のアプリを使う必要がありました．特に不満もなく使用しています．

Open Sync
開発元：Deepen Dhulla
価格：無料
平均評価：3.5（合計106件）
ダウンロード：1万回以上
（2017年12月23日時点）

Just Reminder

リマインダに使っています．やること(洗剤買う・工具持っていく・家賃払うなど)を指定した時刻に教えてくれます．時間になったら通知音と共に全画面で通知をしてくれるので便利です．

ちょうどリマインダー
開発元：AppHouze Co
価格：無料（アプリ内購入あり）（広告を含む）
平均評価：4.5（合計31,469件）
ダウンロード：100万回以上
（2017年12月23日時点）

Solid Explorer Classic

ファイル管理に使っています．オンラインストレージもスムーズに使用することが出来ます．GoogleDrive，DropboxやWebDAVに対応しているので幸せです(ownCloudが使いたかったのでこのアプリを選びました)．画面も2カラムあるため，あるファイルを別のフォルダにコピーするという作業がパパっとできます．
気に入ったので広告なしの有償版を購入しました．

Solid Explorer Classic
開発元：NeatBytes
価格：無料（広告を含む）
平均評価：4.5（合計49,687件）
ダウンロード：100万回以上
（2017年12月23日時点）

FastHub

Github専用のビューワです．ブラウザからできることがほとんどアプリでできるので快適です．

FastHub for GitHub
開発元：Fast Access
価格：無料（アプリ内購入あり）
平均評価：4.7（合計1,316件）
ダウンロード：10万回以上
（2017年12月23日時点）

Termius

sshできます．自鯖がトラブったり，ちょっとした作業がしたい時に活用しています．仮想キーボードも充実しているのでこのアプリがあれば大体のトラブルに対処できます．

Termius - SSH, Mosh and Telnet client
開発元：Crystalnix Limited
価格：無料（アプリ内購入あり）
平均評価：4.7（合計12,361件）
ダウンロード：10万回以上
（2017年12月23日時点）

ExtraVolumeSimple

Androidの音量最小値より小さい音を出したい時に便利です．静かなところでイヤホンで音楽を聞く時などに便利です．

ExtraVolumeSimple(音量微調整)
開発元：Keshigoto
価格：無料（広告を含む）
平均評価：3.9（合計763件）
ダウンロード：5万回以上
（2017年12月23日時点）

Screen Filter

画面の輝度を落とすアプリです．夜行バスの中で携帯を使う時に輝度をかなり絞ることが出来て，意外と有用です．

Screen Filter
開発元：haxor industry
価格：無料
平均評価：4.5（合計123,793件）
ダウンロード：500万回以上
（2017年12月23日時点）

Moneytree

口座残高管理アプリです．このアプリはクレジットの引き落とし日と金額の通知が見やすいので使っています．というかそれにしか使っていません．

家計簿Moneytree - 無料で人気の簡単に節約できるアプリ
開発元：Moneytree KK
価格：無料
平均評価：4.4（合計955件）
ダウンロード：10万回以上
（2017年12月23日時点）

Keepass2Android

パスワード管理アプリです．僕はWindowsでもLinuxでもKeePassを使っているので，このアプリを使うことでAndroidでもパスワードを確認することができます．僕の場合パスワードのデータはWebDAVサーバ上に置いているので，いつでも最新のパスワードデータを確認することが出来ます．

Keepass2Android Password Safe
開発元：Philipp Crocoll (Croco Apps)
価格：無料
平均評価：4.7（合計18,546件）
ダウンロード：100万回以上
（2017年12月23日時点）

AliExpress

言わずと知れたAliExpressのアプリです．アプリ専用の割引があるのでおすすめです．また，毎日コインをもらうことができ，交換で割引クーポンを手に入れられるのでみなさんも是非頑張ってみてください．

AliExpress Shopping App - Coupons For New User
開発元：Alibaba Mobile
価格：無料
平均評価：4.6（合計4,784,908件）
ダウンロード：1億回以上
（2017年12月23日時点）

yakudo camera

誰でも躍動写真が撮影できます．

yakudo camera
